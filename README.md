# 🎯 Java Design Patterns

This repository contains a comprehensive set of classic and modern **design patterns implemented in Java**, organized into three main categories:

- 🛠️ **Creational Patterns**
- 🧱 **Structural Patterns**
- 🔁 **Behavioral Patterns**

Each pattern is implemented in a separate Java file within its category folder, along with a dedicated README for easy understanding and navigation.
